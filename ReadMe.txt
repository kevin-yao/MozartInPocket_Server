This is a server program which connect Mysql database. If you want to run this program on your computer, you should follow below instructions:
1. Modify static variables "oracleURL", "username" , "password" in package "mozartinpocket.database" "DbConnection.java" file with the data based 
on the host url of Mysql in your computer.
2. Run the test program in package mozartinpocket.test to create user, music and post table.
3. Modify the ip in MUSICURL and IMAGEURL in package "mozartinpocket.servlet" "RecievePostServlet.java" file based on the ip in your computer.
4. Run the program on tomcat server.
